pmfow (Package Manager for Old Windows)

pmfow (Package Manager for Old Windows) is a program that allows you to install applications from the command line in older versions of Windows similarly to how you would do it in Linux or in Windows 10 with winget. You can find the Github repository by clicking here.

Requirements

pmfow requires Windows 2000 or later for the 32 bit version and Windows XP x64 or later for the 64 bit version.

You can find the full requirements list by clicking here.

Commands

You can use the following commands with pmfow:

  • pmfow install: Installs a package
  • pmfow uninstall (only in the development version for now): Uninstalls a package
  • pmfow update: Updates the repositories
  • pmfow list: Lists all the packages that can be installed
  • pmfow search: Searches for a package
  • pmfow version: Shows information about the version of pmfow you are running
